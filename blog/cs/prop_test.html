Data Structure

A data structure is a collection of aggregate of data. The data may be binary values, numbers, characters, or data structure.

the following are basic data structure.

bunch
unpackaged, unindexed
string
unpackaged, indexed
set
packaged, unindexed
array
packaged, indexed

Indexed

elements in a collection can be indexed by binary values, numbers, characters, strings.

a arrary is indexed by positive interger numbers.

Packaged

A bunch of data packaged in a container, such as array, tree.

A packaged data structure can have circular reference, which means it contains itself or two data structure reference each other as a loop. 

var a = [1];
a.push(a);
console.log(a);

In another word, an unpackaged data structure can not have circular reference, a string can not contain itself or another strings.
